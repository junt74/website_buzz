F/A Sound Creation Navigator

所有プラグインを駆使した作曲戦略ガイド

基本理念：サウンドの核

1. 攻撃性 (Aggression)

硬質で高速なリズム、インダストリアルなノイズ、歪んだベースライン、破壊的な効果音。

2. 叙情性 (Melancholy)

哀愁を帯びたキャッチーなメロディ、壮大なストリングスやパッド、空間的な広がり。

1. ドラム：楽曲の骨格とエンジン

プライマリ選択肢 (核となるビート)

Drumazon 2 (TR-909): 主役。BPM150〜165で硬質な4つ打ちキックを。
TAL-Drum: Drumazon 2にレイヤーして厚みを出す。特にクラップやリムショット。

レイヤーと質感 (厚みと個性)

DMX / IconDrum: "いなたい"スネアやタムを薄く重ね、時代感を演出。
XO / Battery 4: インダストリアル系サンプル（金属音など）を追加し、無機質感をプラス。

バスプロセッシング (グルーヴの生成)

Drum Bus → Smasher / 1176 → RX950 / RC-20 → Oxford TransMod

1. 潰して叩く → 2. 汚して劣化させる → 3. アタックを強調

2. ベースライン：グルーヴの駆動力

プライマリ選択肢 (メインのシーケンス)

TAL-BassLine-101 (SH-101): 王道。16分音符でうねるシーケンスを作成。フィルターエンベロープが鍵。
Phoscyon2 (TB-303): ディストーションをかけ、攻撃的で歪んだシーケンスベースに。

レイヤーと補強

Serum / Massive: シンプルなサイン波で超低域を補強。
FM8 / RYM2612: FMシンセの金属的なアタックで輪郭を際立たせる。

プロセッシング

歪み (Lifeline Dirt / Faturator): サチュレーションを加え、サウンドを太く、攻撃的に。
サイドチェイン (Kickstart): キックに合わせてダッキングさせ、グルーヴを生む。

3. リードメロディ：楽曲の「歌」

音色選択 (哀愁とパワー)

U-NO-LX (Juno-60): 温かみのあるノコギリ波で哀愁漂うメロディに。JURA Chorusも活用。
Dreamsynth / PPG Wave 3: SF的な世界観に合う、金属的で冷たい質感のリード。
ReDominator (Alpha Juno): 「Hoover」サウンドでテンションを最高潮に。
Discovery (Nord Lead 2): 硬質でパワフルなサウンド。ボス戦などの緊迫した場面に。

エフェクト処理 (空間と感情)

ディレイ (Comeback Kid / EchoBoyJr): 符点ディレイで空間と緊張感を演出。
リバーブ (ValhallaVintageVerb / LX480): 80年代風のチープな響きや、リッチなホールで壮大さを加える。

4. コード / パッド / ヒット：世界観とドラマの構築

パッド / ストリングス

Polymode / GX-80: 壮大で豊かなパッドやストリングス。フィルターのオートメーションで展開を劇的に。

スタブ / ヒット (アクセントと衝撃)

M1 / Nexus / Orbhits Plus: 「オーケストラルヒット」の宝庫。
Emulation II+ / DarkLight CMI: 80年代特有のサンプリングされたボイスやブラスヒット。
Rave Generator3: 90年代初頭のレイヴ風コードスタブを生成。

プロセッシング (時代感の統一)

Super VHS / RC-20: Lo-Fiな質感を積極的に加え、全体の時代感を統一する。

5. SE / テクスチャ：破壊と混沌の演出

CA2600 / Voltage Modular: 予測不能なノイズ、金属的なサウンド、爆発音などのSE作成に最適。
Effectrix / Turnado: リアルタイムでグリッチ、スタッター、リバースなどの破壊的なエフェクトを追加。
Chipsynthシリーズ / miniBit: レトロゲーム風のシンプルな効果音（爆発音、アラート音）をアクセントに。

6. ミックス & マスタリング戦略

1. チャンネルストリップでの音作り

SSL Native Essentials: クリアでモダンなパンチを付加。
API Vision Channel Strip: より音楽的で温かみのあるキャラクターを付加。

2. バスコンプでの接着

bx_Townhouse Buss Compressor: ドラムバスやマスターバスにかけ、全体を軽く接着し一体感を出す。

3. 最終仕上げ (マスタリング)

Master Bus → Ozone 11 → Inflator / Sausage Fattener → Elephant / TDR Limiter

常にSPAN Plusで周波数バランスを確認。

このガイドは、お持ちのプラグインリストを基に「F/A」風サウンドの構築戦略を提示するものです。

最高の戦略は実験と創造の中にあります。Enjoy your music production!
