ジェットファンク：長沼英樹サウンド再現プラグイン構成ガイド

長沼英樹氏の楽曲スタイルは、テクノ、ファンク、ソウル、ヒップホップを融合したユニークなサウンドが特徴です。このスタイルを「ジェットファンク」と名付け、その魅力をDTMで再現するためのプラグイン構成ガイドをまとめました。お手持ちのプラグインを効果的に活用することで、この「ジェットファンク」サウンドをあなたの制作に取り入れることが可能です。

ジェットファンクサウンドの主要要素

ジェットファンクサウンドを構成する主な要素は以下の通りです。

  • サンプリングの芸術: ファンク、ジャズ、ディスコ、ヒップホップなどからのサンプルを巧みに取り入れ、ボーカルやその他の音源を実験的に加工。
  • 速いBPMのファンクグルーヴ: エネルギッシュなペースとリズミカルな特性を持つドライビングビートとシンセサイザーサウンド。ブレイクビーツやEDMの要素も多く取り入れられる。
  • アシッドベースの導入: Roland TB-303のような「うねるような、催眠的なサウンド」。
  • 「声ネタ」の活用: 声のサンプルをカット、再配置、加工し、ユニークなアクセントとテクスチャを加える。

プラグイン構成ガイド

1. ドラムとグルーヴの構築

ジェットファンクサウンドの核となる「速いBPMのファンクグルーヴ」と「ブレイクビーツ」を構築します。

  • ドラムサウンドの選択:
    • TAL Drum / Battery 4 / XO: ファンクやブレイクビーツに適したドラムサンプルを選びます。TAL Drumは80-90年代のドラムマシンサウンドに特化しており、レトロな質感に合致します。Battery 4やXOは幅広いドラムサウンドから選べます。
    • DMX / IconDrum: 80年代のヒップホップやポップスで使われたOberheim DMXやLinnDrumのエミュレーションは、ジェットファンクのルーツとなるファンクやソウルのドラムサウンドに近づけるのに有効です。
    • RX1200: E-mu SP-1200の12bitサンプラーサウンドを再現。キックやスネアに適用すると、ヒップホップ的なクランチ感のあるローファイな質感が得られます。
  • グルーヴの強化:
    • ShaperBox / Effectrix / Turnado: ドラムループやブレイクビーツにこれらのマルチエフェクトを適用し、グリッチ、スタッター、フィルターモジュレーションなどを加えることで、予測不能でエネルギッシュなグルーヴを演出します。
    • HalfTime: ドラムループ全体や特定のドラムパートに適用し、ハーフスピードのエフェクトを加えることで、独特の浮遊感や重厚感を出すことができます。

2. サンプリングと声ネタの活用

ジェットファンクサウンドの真骨頂である「サンプリングの芸術」と「声ネタの活用」に、TAL-SamplerとAmigo Samplerを積極的に使用します。

  • TAL-Samplerの活用:
    • ビンテージサンプラーの質感: ファンク、ジャズ、ディスコ、ソウル、ヒップホップなどの既存の楽曲からボーカル、ホーン、ギター、シンセなどのフレーズをサンプリングし、TAL-Samplerで加工します。
    • 実験的な加工: サンプルを大きくピッチシフトしたり、タイムストレッチして、元の意味が失われるまで加工することで、長沼氏の実験的なアプローチを再現します。
    • ローファイ設定: TAL-SamplerのDA/ADコンバーターエミュレーションやサンプリングレート/ビットデプス設定を調整し、意図的に音質を劣化させ、独特のローファイな質感を付加します。
    • ループとチョップ: サンプルを細かくチョップし、新しいフレーズとして再構築したり、短いループを作成してリズミカルな要素として組み込んだりします。
  • Amigo Samplerの活用:
    • 素早いサンプリングと演奏: インスピレーションが湧いたときにすぐに音源を取り込み、試行錯誤するのに適しています。
    • 声ネタの即興性: 「声ネタ」をAmigo Samplerに読み込み、リアルタイムでトリガーしたり、ピッチを変えたり、短いフレーズとしてループさせたりすることで、即興的なパフォーマンス感を加えることができます。
    • ワンショットサンプルの活用: ドラムのワンショットサンプルや、効果音などをAmigo Samplerに読み込み、直感的に配置してグルーヴを補強します。
  • その他のサンプラー/音源:
    • Emulation II + / DarkLight CMI: 80年代の象徴的なサンプラーのエミュレーションは、当時のデジタルサウンドの質感を再現するのに非常に有効です。
    • HERA-ONE: 女性ボーカルチョップに特化した音源なので、声ネタのバリエーションを増やすのに役立ちます。

3. アシッドベースとシンセサウンド

ジェットファンクサウンドの特徴である「アシッドベース」と、ファンキーでエネルギッシュなシンセサウンドを構築します。

  • アシッドベース:
    • Phoscyon2 / TAL BassLine 101 / DeeACID: Roland TB-303やSH-101のエミュレーションは、アシッドベースラインの作成に最適です。フィルターのカットオフとレゾナンスをオートメーションで動的に変化させ、「うねり」と「歪み」を強調します。
    • LowSane: 低域に特化したディストーション/マルチエフェクトなので、アシッドベースをさらにアグレッシブにしたり、太さを加えたりするのに活用できます。
  • シンセリード/パッド/SE:
    • U-NO-LX / GX-80 / PolyMAX / ReDominator / Super8: 温かみのあるアナログサウンドや、80年代のポリシンセサウンドを再現するのに最適です。
    • Dreamsynth / BA-1 / ATOMIKA: ユニークな質感のシンセサウンドを作るのに役立ちます。
    • Discovery: パワフルなリードやベース、SEに最適です。
    • FM8 / Chipsynth MD / RYM2612: きらびやかなベル、エレピ、複雑なベース、そしてジェットファンクサウンドにも見られるような独特の質感のサウンドを作るのに非常に有効です。チップチューンシンセは、レトロゲームのようなSEやメロディに活用できます。
    • Serum / Massive / SPIRE / Sylenth1: 現代的なEDMやダンスミュージックの要素を取り入れたい場合に活用できます。
    • Rave Generator3: 90年代のレイヴミュージック特有のスタブやフーバーサウンドを生成できます。

4. エフェクト処理

ジェットファンクサウンドの「質感」や「空気感」を再現するために、エフェクトを積極的に使用します。

  • ローファイ/ビンテージ質感:
    • RC-20 / Super VHS / BABY AUDIO TAIP / RX950 / MSXII Lo-fly Dirt / Lifeline Dirt / Tape / Magnetite / DirtyTape / Flavor Pro / PCM2612: これらのプラグインは、レコードノイズ、テープの揺れ、サチュレーション、ビットクラッシュなど、様々な形でローファイな質感やビンテージ感を付加できます。あらゆるトラックに積極的に適用し、ジェットファンクサウンド特有の「汚れた」しかし「温かい」質感を追求します。
    • GameVerb: 8bit/16bitゲーム風のチープで個性的なリバーブは、ジェットファンクサウンドのレトロゲーム的な雰囲気を強調するのに役立ちます。
  • ダイナミクス/ミキシング:
    • SSL Native Essentials Bundle / API Vision Channel Strip / bx_Console N / 32Classic Channel: これらのチャンネルストリップは、ジェットファンクサウンドのパンチとクリアさを両立させるのに役立ちます。
    • UAD 1176 / bx_Townhouse Buss Compressor / Smasher / Combustor / HAMMER / VINAI XTT / DynaBunny: 各種コンプレッサーは、ドラムやベースをタイトにしたり、アグレッシブなサウンドにしたりするのに活用します。
    • Oxford Inflator / DeePopMax: 音圧とラウドネスを自然に向上させ、楽曲全体に迫力を与えます。
    • Oxford TransMod: ドラムのアタックを強調したり、シンセのサステインを調整したりして、グルーヴと音の立ち上がりをコントロールします。
  • 空間系:
    • ValhallaVintageVerb / LX480 Essentials / Echorec / Comeback Kid / EchoBoyJr: リバーブやディレイは、サウンドに深みや広がりを与えるだけでなく、ジェットファンクサウンドの持つ「浮遊感」や「サイケデリックな雰囲気」を演出するのに重要です。
  • ボーカル処理:
    • VocalSynth2 / DeeVocoder / Nectar3: 声ネタやSynthesizer Vで生成したボーカルに、ボコーダーやトークボックス、ハーモナイザーなどのエフェクトを適用し、長沼氏のように実験的に加工します。

全体的なワークフローのヒント

  • サンプルの選定と加工: ファンク、ジャズ、ソウル、ヒップホップなどからフレーズをサンプリングし、TAL-SamplerやAmigo Samplerでピッチシフト、タイムストレッチ、チョップ、リバース、ローファイ加工などを徹底的に行います。
  • ドラムとベースライン: 速いBPMのブレイクビーツやファンクグルーヴを構築。ドラムにはRX1200やTAL Drumでローファイ感を加え、アシッドベースにはPhoscyon2やTAL BassLine 101で「うねり」と「歪み」を強調します。
  • シンセレイヤー: アナログシンセやFMシンセで、ファンキーなリード、分厚いパッド、特徴的なSEなどを加えます。チップチューンシンセでレトロなSEやメロディを挟むのも効果的です。
  • エフェクトによる質感作り: 各トラックやバスにRC-20、Super VHS、BABY AUDIO TAIPなどのローファイエフェクトを積極的に適用し、ジェットファンクサウンド特有の「汚れた」しかし「温かい」質感を追求します。
  • ミキシングとマスタリング: チャンネルストリップやコンプレッサーでサウンドを整え、Ozone11やElephantでマスタリングし、音圧とクリアさを両立させます。
